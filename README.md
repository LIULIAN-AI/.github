# .github

LIULIAN organization profile repo. The actual profile content GitHub
renders on the org page is at [`profile/README.md`](profile/README.md).

Why this repo exists:
- `profile/README.md` is GitHub's special-case path — anything at
  `liulian-ai/.github/profile/README.md` shows on the org landing
  page above the repo grid.
- The `profile/assets/` folder holds screenshots referenced by the
  profile README (served via `raw.githubusercontent.com`).
